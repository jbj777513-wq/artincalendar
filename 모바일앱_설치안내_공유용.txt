📱 아트인캘린더 모바일 앱 설치 안내

팀 일정을 휴대폰에서 실시간으로 보고 추가할 수 있는 앱입니다.
앱스토어에서 받을 필요 없이, 아래 주소만 열면 바로 사용할 수 있어요.

────────────────────────────
🌐 주소
https://jbj777513-wq.github.io/artincalendar/
────────────────────────────

이 주소를 휴대폰 브라우저에서 열면 바로 캘린더가 뜹니다.
(우리 팀 일정이 자동으로 함께 보입니다 — 따로 설정할 게 없어요.)

매번 주소를 치기 번거로우니, 아래처럼 "홈 화면에 추가"하면
일반 앱처럼 아이콘으로 바로 열 수 있습니다.


■ 아이폰 (iPhone) — Safari 사용
 1. Safari 브라우저로 위 주소 접속
 2. 화면 아래쪽 '공유' 버튼( □↑ 모양 ) 누르기
 3. 메뉴에서 '홈 화면에 추가' 선택
 4. 오른쪽 위 '추가' 누르기
 → 홈 화면에 아트인캘린더 아이콘 생성 완료!

 ※ 꼭 Safari로 열어주세요. (크롬 등 다른 브라우저는 홈 화면 추가가 안 될 수 있어요)


■ 안드로이드 (삼성/갤럭시 등) — Chrome 사용
 1. Chrome 브라우저로 위 주소 접속
 2. 오른쪽 위 점 3개( ⋮ ) 메뉴 누르기
 3. '홈 화면에 추가' 또는 '앱 설치' 선택
 4. '추가' 누르기
 → 홈 화면에 아트인캘린더 아이콘 생성 완료!


■ 사용 방법
 · 날짜를 누르면 그날 일정 보기 / 추가 / 수정 가능
 · 위쪽 ☰ 버튼을 누르면 '다가오는 일정'을 목록으로 한눈에 보기
 · 좌우로 밀면 이전/다음 달로 이동
 · 추가한 일정은 다른 팀원과 PC에도 실시간으로 함께 표시됩니다


■ 참고
 · 완전 무료이며, 업데이트도 자동으로 됩니다 (다시 설치할 필요 없음)
 · 인터넷이 연결되어 있어야 다른 사람과 일정이 공유됩니다
 · 화면 위쪽의 초록색 점이 켜져 있으면 정상 연결 상태입니다
