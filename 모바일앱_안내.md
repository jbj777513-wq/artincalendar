# 📱 아트인캘린더 모바일 앱 (PWA) 안내

기존 PC 앱과 **같은 Firebase 데이터를 실시간 공유**하는 모바일 웹앱입니다.
iOS·안드로이드 모두 사용 가능하며, 별도 앱스토어 등록 없이 홈 화면에 설치합니다.

## 🌐 주소
**https://jbj777513-wq.github.io/artincalendar/**

휴대폰 브라우저에서 위 주소를 열면 바로 캘린더가 뜹니다.
(기본 그룹 `ARTIN` 으로 접속 → PC 앱과 같은 일정이 보입니다.)

---

## 📲 휴대폰에 "앱"으로 설치하기

### 아이폰 (Safari)
1. Safari로 위 주소 접속
2. 하단 **공유 버튼**(□↑) 탭
3. **"홈 화면에 추가"** 선택 → 추가
4. 홈 화면에 아트인캘린더 아이콘 생성 ✅

### 안드로이드 (Chrome)
1. Chrome으로 위 주소 접속
2. 우측 상단 **⋮ 메뉴** 탭
3. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택 ✅

설치하면 주소창 없는 전체화면 앱처럼 실행됩니다.

## 🔄 자동 업데이트 (요청하신 핵심 기능)

코드를 수정하고 `webapp/` 폴더 변경분을 **GitHub main에 push**하면:

```
push → GitHub Actions 자동 배포(약 30초) → 사용자가 앱을 다시 열면 자동으로 최신 버전
```

- 앱스토어 심사·재설치 **불필요**
- 서비스워커가 항상 최신본을 받아오고, 새 버전 감지 시 자동 새로고침
- 버전 번호는 배포할 때마다 자동으로 올라감 (`1.0.N`)

### 배포 방법
1. 클로드에게 수정 요청 (예: "다음 달 버튼 색 바꿔줘")
2. 변경 후 push:
   ```
   git add webapp && git commit -m "수정 내용" && git push
   ```
3. 끝 → 잠시 후 휴대폰에서 앱 다시 열면 반영됨

배포 진행 확인: https://github.com/jbj777513-wq/artincalendar/actions

## 🎨 아이콘
흰색 배경 + 남색 AIS 로고 (기존 `logo_white.png`를 남색으로 변환).
로고를 바꾸려면 `logo_white.png` 교체 후 `webapp/generate_icons.py` 실행 → 재배포.
